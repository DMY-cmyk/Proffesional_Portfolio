The Effect of Sustainability Report Disclosure on Firm Value in the Manufacturing Sector

Panel-data analysis of Indonesian manufacturers listed on IDX. The study tests whether disclosure quality, measured against GRI Standards, is priced by investors — using Tobin's Q and market-to-book as firm-value proxies, controlling for leverage, size, and profitability.
