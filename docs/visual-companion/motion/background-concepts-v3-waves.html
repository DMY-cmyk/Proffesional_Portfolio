What changed, concretely
1 · DriftEach wave has a horizontal offsetX that increases continuously. Different speeds per layer (back: 8 px/s, front: 22 px/s) gives parallax depth.
2 · PhaseInstead of translateY, each wave's y-position is computed per-x using sin(kx + ωt). So peaks and troughs move independently along the wave — not as a rigid unit.
3 · BreathingAmplitude is modulated by a slow second sine: A · (1 + 0.3 sin(0.1t)). The wave height gently swells and relaxes.
4 · CursorFor points near the cursor, we add a soft gaussian bulge toward the cursor's y (max 28px). Effect decays over ~180px radius.
5 · TicksThree accent circles ride along the wave — their x advances with the wave drift; their y snaps to the wave's current shape. Opacity pulses.
Cost~4 × 80 = 320 line segments per frame + 3 circle draws. Under 0.5% CPU on a modest laptop. Respects prefers-reduced-motion → freezes to a static snapshot.
