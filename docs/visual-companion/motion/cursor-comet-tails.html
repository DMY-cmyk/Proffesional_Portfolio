Cursor comet tails — 3 concepts · Option B as base

Each panel renders Option B's teal ring + center dot plus a different tail. Move your mouse inside each panel — move slow, fast, in spirals, over the buttons. The native cursor is hidden inside each stage so you see only the custom cursor.

Option A

Particle fade

~15 small teal dots at recent cursor positions. Each dot shrinks and fades over 400ms. Tail length is constant regardless of speed. Classic "magic wand."

Option B · Recommended

Gradient streak (speed-reactive)

A single elongated gradient shape behind the cursor. Length scales with velocity: slow = ~30px, fast swipe = ~140px. Gold accent at the tip. Feels like an actual comet.

Option C

Painted curve (brushstroke)

Tapered smooth curve through last 15 cursor positions. Width fades 2.5→0.5px. Color: teal → gold → transparent. Follows the shape of your path — draw a spiral and see.

Pick one
A — ParticleSafest, immediately legible. Tail length constant — feels controlled rather than comet-like.
B — StreakMy pick. Honors the word "comet" — tail grows with speed. Most dynamic and kinetic feel. Gold tip pops against teal body.
C — Painted curveMost distinctive and artistic. Best when the user draws. Risk: looks like a brushstroke when used heavily, not a comet.
GlobalAll three hide on prefers-reduced-motion and on touch devices (falls back to native cursor).
